# ID란?
입력/출력을 의미. 파일 입출력 콘솔입출력 .. 등등 
입출력의 종류는 다양하지만 , 입출력이라는 공통점이 있다!
JAVA에서는 이런 입출력의 공통적인 부분을 인터페이스로 추상화하고 있다.
가령 wirte의 매서드는 동일하지만 어떤식으로 동작하는지는 입/출력의 종류마다 다르게 된다.

장치(키보드/파일/USB/네트워크..) ---> 입/출력 ----> 인터페이스 ---> 구현 -----> 구현체()
